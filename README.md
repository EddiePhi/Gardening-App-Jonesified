# Plot-Of-Gold


Description:
    Plot of Gold provides a quick and easy hub for all your gardening needs.

Features:

    Plot Generator

    Forecast Info

    Plant info

Technologies used:
    
    Front End: HTML5, CSS3, Bulma CSS, NES.CSS,
    
    Back End: JavaScript, jQuery, NPM (Sequelize, Fetch, dotenv)
    
    Database: mySQL
    
    External API: https://openweathermap.org/
